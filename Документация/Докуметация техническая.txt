///Переменные классов и передаваемые перенные для MainWindow///

1. self.ACTIVE_MOD: str | None = None
	показывает в каком режиме используется приложение
	принимает значение: 
		'bd'- рабрта с базой данных
		'excel'- работа с экселем
		None - не выбран режим
		с данной переменной использем синтаксическую конструкцию math case, где case выше представленные принимаемые значения

2. self.SAVE_DATA_MODE: bool = True
	может быть True и False
	показывает сохранён ли пользовательский результат
	изменяется на False при следующих операциях:
		добавление значений и измерений
		удаление значений и измерений
		изменение значений

	изменяется на True при следующих операциях:
		сохранение ctrl + s
		сохранить как ctrl + shift + s

3. self.DATA: dict = dict()
	хранит значение выводящиеся на экран и заполняющие все виджеты и комбо боксы
	формат {(1, Измерения бетон 12-b, True, (113, 228)): [12, 12, 12, 12, 12, 12, 12, 12, 113, 228]}
	
	ключ (1, Измерения бетон 12-b, True, (113, 228)) на 1 позиции номер столбца или id измерения,
	2 позиция название, 3 позиция состояние расчитан результат или нет если True на 4 позиции расчитанные значения 
	
	значения [12, 12, 12, 12, 12, 12, 12, 12, 113, 228] - значения полученные в результате измерений

4. self.EXCEL_PATH: str | None = None
	хранит путь до открытого экселевского файла.
	имеет вид "D:\Calculate\test.xlsx"

5. self.CHANGE_MODE = False
	при двойном нажатии по элементу в listWidget меняется на True
	при совершённом изменении меняется на False

6. full_data
	переменная использующая технологию Signal из класса(окна) AddDialog
	передаёт словать стандартного формата без данных столбца и результата расчёта.
	
7. read_excel_signal
	переменная использующая технологию pyqtSignal для расчёта параллельно основного окна(ядро: ReadThread)
	после передачи записывантся в self.DATA
